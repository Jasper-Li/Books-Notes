#### 学习 Java 和 Smali 的练习题
本文件夹里面，包含了我学习 Java 和 Smali 的练习题, 题目选自《Core.Java练习题与编程规范》，在 doc-markdown 文件夹里。
Smali 是 Java 在 Android Dalvik 虚拟机里的“汇编语言”方式。
我学习的方式如下：
	1. 用 Java 编写小的程序
	2. 编译成 dex 文件
	3. 反编译 dex 文件成 smali 汇编格式
	4. 分析 smali 文件

####文件夹说明
	1. code：练习用的 Java 和 反编译的 Smali 代码。
	2. doc-markdown: markdown 标记语言格式的说明文档，描述 samli 语言的分析方法和分析结果。复杂的有带流程图。
	3. sources: 某些图片源文件，如流程图等。
